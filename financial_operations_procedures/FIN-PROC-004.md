# AML Program Independent Testing Procedure (FIN-PROC-004)

### 1. Purpose

To define the process for conducting annual independent testing of **[Company Name]**'s Anti-Money Laundering (AML) program as required by BSA Pillar 2, ensuring program effectiveness and identifying areas for improvement.

### 2. Scope

This procedure applies to:

- Annual independent testing of all BSA/AML program elements
- Testing by internal audit or qualified third party
- Remediation of identified findings
- Reporting to senior management and board

### 3. Overview

This procedure establishes the workflow for annual independent testing of the AML program per BSA requirements and FIN-POL-001. Testing covers all five pillars and program components to assess effectiveness and compliance.

### 4. Procedure

#### 4.1 Test Planning

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Internal Audit / Third Party | Develops annual AML test plan based on: |
| | | - Regulatory requirements |
| | | - Previous test findings |
| | | - Risk assessment results |
| | | - Regulatory examination findings |
| | | - Changes to products or services |
| **2** | Internal Audit / Third Party | Defines testing scope including: |
| | | - Transaction monitoring |
| | | - SAR process |
| | | - CIP/CDD |
| | | - OFAC compliance |
| | | - Training program |
| | | - Recordkeeping |
| **3** | BSA/AML Officer | Reviews and provides input on test plan. |
| **4** | Internal Audit / Third Party | Finalizes test plan and timeline. |

#### 4.2 Testing Execution

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Internal Audit / Third Party | Conducts testing activities: |
| | | - Transaction sampling and review |
| | | - SAR file review |
| | | - CIP documentation review |
| | | - Policy and procedure review |
| | | - Staff interviews |
| | | - System testing |
| **2** | Internal Audit / Third Party | Evaluates transaction monitoring effectiveness: |
| | | - Alert generation appropriateness |
| | | - Alert disposition quality |
| | | - SAR decision accuracy |
| **3** | Internal Audit / Third Party | Evaluates SAR process: |
| | | - Timeliness of filing |
| | | - Narrative quality |
| | | - Decision documentation |
| **4** | Internal Audit / Third Party | Evaluates CIP/CDD: |
| | | - Identity verification adequacy |
| | | - Beneficial ownership collection |
| | | - Risk rating accuracy |
| | | - EDD appropriateness |
| **5** | Internal Audit / Third Party | Evaluates training: |
| | | - Training completion rates |
| | | - Training content adequacy |
| | | - Role-based training appropriateness |
| **6** | Internal Audit / Third Party | Documents findings and observations. |

#### 4.3 Reporting

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Internal Audit / Third Party | Prepares written report including: |
| | | - Scope of testing |
| | | - Methodology |
| | | - Findings and severity ratings |
| | | - Recommendations |
| | | - Management response section |
| **2** | BSA/AML Officer | Reviews draft report and provides management response. |
| **3** | Internal Audit / Third Party | Finalizes report with management responses. |
| **4** | BSA/AML Officer | Presents findings to senior management. |
| **5** | Senior Management / Board | Reviews and acknowledges AML test results. |

#### 4.4 Remediation

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | BSA/AML Officer | Develops remediation plan for each finding with: |
| | | - Specific action items |
| | | - Responsible parties |
| | | - Target completion dates |
| **2** | Responsible Parties | Implement remediation actions. |
| **3** | BSA/AML Officer | Tracks remediation progress. |
| **4** | Internal Audit / Third Party | Validates remediation of significant findings. |
| **5** | BSA/AML Officer | Reports remediation status to board **[Frequency, e.g., quarterly]**. |

#### 4.5 Tester Qualifications

| **Requirement** | **Description** |
|-----------------|-----------------|
| Independence | Tester independent from AML program operations |
| Qualifications | CAMS certification or equivalent experience |
| Knowledge | Understanding of BSA/AML requirements |
| Experience | Prior AML testing or examination experience |
| Objectivity | No conflicts of interest |

### 5. Standards Compliance

| **Procedure Step(s)** | **Standard/Framework** | **Control Reference** |
|-----------------------|------------------------|-----------------------|
| **4.1-4.5** | BSA Five Pillars | Pillar 2 - Independent Testing |
| **4.1-4.5** | FFIEC BSA Manual | Independent Testing section |
| **4.3** | SOC 2 TSC | CC1.2 |

### 6. Artifact(s)

- AML test plan
- Testing workpapers
- Final test report
- Management responses
- Remediation tracking documentation
- Board presentation/acknowledgment

### 7. Definitions

| **Term** | **Definition** |
|----------|----------------|
| **Independent Testing** | Testing performed by qualified internal audit or third party independent of AML program operations. |
| **CAMS** | Certified Anti-Money Laundering Specialist - professional certification in AML. |
| **Finding** | An identified deficiency or area requiring improvement in the AML program. |

### 8. Responsibilities

| **Role** | **Responsibility** |
|----------|-------------------|
| **Internal Audit / Third Party** | Plan and execute testing. Document findings. Prepare report. Validate remediation. |
| **BSA/AML Officer** | Provide testing access. Develop remediation plans. Track progress. Report to board. |
| **Compliance Team** | Support testing activities. Implement remediation. |
| **Senior Management / Board** | Review test results. Ensure adequate resources for remediation. |
