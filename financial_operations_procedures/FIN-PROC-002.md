# Suspicious Activity Reporting (SAR) Procedure (FIN-PROC-002)

### 1. Purpose

To define the process for identifying, investigating, and reporting suspicious activity to FinCEN as required by the Bank Secrecy Act, ensuring timely and accurate SAR filing per 31 CFR 1020.320.

### 2. Scope

This procedure applies to:

- All suspicious activity detected through transaction monitoring
- All suspicious activity reported by employees
- All suspicious activity indicators in customer relationships
- All SAR filing decisions and documentation

### 3. Overview

This procedure establishes the workflow for suspicious activity investigation and SAR filing. It covers alert review, investigation, SAR decision-making, filing, and recordkeeping per BSA requirements and FIN-POL-001.

### 4. Procedure

#### 4.1 Suspicious Activity Detection

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Transaction Monitoring System | Generates alert based on rule triggers or anomaly detection. |
| **2** | Employee / Customer Contact | Alternatively, reports suspicious behavior or activity via internal reporting channel. |
| **3** | Operations Team | Documents any customer-facing suspicious indicators. |
| **4** | Compliance System | Routes alert or report to Compliance Team for review. |

#### 4.2 Alert Review and Investigation

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Compliance Analyst | Reviews alert details including: |
| | | - Transaction(s) that triggered the alert |
| | | - Customer profile and history |
| | | - Alert rule or reason |
| **2** | Compliance Analyst | Conducts investigation: |
| | | - Reviews transaction history for patterns |
| | | - Reviews customer documentation |
| | | - Checks for previous SARs or alerts |
| | | - Conducts open-source research if needed |
| | | - Documents all investigative steps |
| **3** | Compliance Analyst | Determines if activity is: |
| | | - Explainable and not suspicious → Close alert |
| | | - Potentially suspicious → Continue investigation |
| | | - Clearly suspicious → Prepare SAR recommendation |
| **4** | Compliance Analyst | If potentially suspicious, gathers additional information: |
| | | - Request information from business unit (if no tip-off risk) |
| | | - Additional database searches |
| | | - Document all findings |
| **5** | Compliance Analyst | Prepares investigation summary and recommendation for SAR decision. |

#### 4.3 SAR Decision

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Compliance Analyst | Submits investigation summary to BSA/AML Officer with recommendation: |
| | | - SAR recommended: Include narrative justification |
| | | - No SAR: Include rationale for non-filing |
| **2** | BSA/AML Officer | Reviews investigation and recommendation. |
| **3** | BSA/AML Officer | Makes SAR decision considering: |
| | | - Is there known or suspected violation of law? |
| | | - Does transaction involve $5,000+ and suspect is known? |
| | | - Does transaction involve $25,000+ regardless of suspect? |
| | | - Does activity have no business or lawful purpose? |
| **4** | BSA/AML Officer | Documents SAR decision and rationale. |
| **5** | Compliance Analyst | If SAR approved, proceeds to SAR preparation. |
| **6** | Compliance Analyst | If no SAR, closes case with documentation. |

#### 4.4 SAR Preparation and Filing

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Compliance Analyst | Prepares SAR using FinCEN SAR form, including: |
| | | - Subject information (if known) |
| | | - Financial institution information |
| | | - Suspicious activity information |
| | | - Narrative describing activity |
| **2** | Compliance Analyst | Drafts narrative including the five W's: |
| | | - Who is conducting the activity |
| | | - What instruments or mechanisms are involved |
| | | - When did the activity occur |
| | | - Where did the activity take place |
| | | - Why is the activity suspicious |
| **3** | BSA/AML Officer | Reviews and approves SAR before filing. |
| **4** | BSA/AML Officer or Designee | Files SAR through BSA E-Filing system. |
| **5** | Compliance System | Records SAR filing confirmation number and date. |

#### 4.5 SAR Timing Requirements

| **Situation** | **Filing Deadline** |
|---------------|---------------------|
| Suspect identified | 30 calendar days from detection |
| No suspect identified | 60 calendar days from detection (with documented attempts to identify) |
| Ongoing activity | Continuing SARs every 90 days as appropriate |
| Imminent threat | Immediately notify law enforcement; SAR filed within normal timeframe |

#### 4.6 Recordkeeping and Confidentiality

| **Step** | **Who** | **What** |
|----------|---------|----------|
| **1** | Compliance Team | Maintains SAR and supporting documentation for 5 years from filing. |
| **2** | All Personnel | Maintains strict confidentiality of SAR filing. |
| **3** | BSA/AML Officer | Ensures no disclosure of SAR to subject ("no tipping"). |
| **4** | Compliance Team | Provides SAR information to law enforcement upon proper request (314(a), subpoena). |

### 6. Artifact(s)

- Alert documentation
- Investigation notes and supporting documents
- SAR decision documentation
- Filed SAR with confirmation number
- Continuing SAR schedule (if applicable)
- 314(a) responses (if applicable)

### 7. Definitions

| **Term** | **Definition** |
|----------|----------------|
| **Suspicious Activity Report (SAR)** | A report filed with FinCEN when suspicious activity is detected that may involve money laundering or other crimes. |
| **314(a) Request** | A FinCEN request for financial institutions to search records for subjects of law enforcement investigations. |
| **Safe Harbor** | Legal protection for financial institutions that file SARs in good faith. |
| **Tipping** | Prohibited disclosure to a SAR subject that a SAR has been or may be filed. |

### 8. Responsibilities

| **Role** | **Responsibility** |
|----------|-------------------|
| **Compliance Analyst** | Review alerts, conduct investigations, prepare SARs, maintain documentation. |
| **BSA/AML Officer** | Make SAR decisions, approve SAR filings, ensure confidentiality. |
| **Operations Team** | Report suspicious customer behavior, support investigations. |
| **All Employees** | Report suspicious activity, maintain SAR confidentiality. |
